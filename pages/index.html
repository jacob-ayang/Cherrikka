CHERRIKKA

Cherry Studio ↔ RikkaHub 备份互转

主题

快速入口

在线版本： https://uwp.de5.net

仓库地址： jacob-ayang/Cherrikka

上手说明（前端）

  1. 上传备份 ZIP。
  2. 选择源格式（auto/cherry/rikka）。
  3. 点击 Convert，下载转换结果。

Fork 自部署（Vercel / Pages）

  1. 先 Fork 本项目到你自己的 GitHub 账号。
  2. 在 Vercel 里导入仓库并完成部署。
  3. 或者启用 GitHub Pages 发布本目录静态页。

Deploy with Vercel

CLI 快速命令

go build -o cherrikka ./cmd/cherrikka
./cherrikka convert \
  --input input.zip \
  --output output.zip \
  --from auto \
  --to rikka
